1. Definition

K-Means is an unsupervised machine learning algorithm that partitions a dataset into K clusters by minimizing the distance between points and their cluster centers (centroids).

2. Steps of K-Means Algorithm

  • Choose K ‚Äì Select the number of clusters you want to create.
  • Initialize centroids ‚Äì Randomly select K points as initial centroids.
  • Assign points ‚Äì Assign each data point to the nearest centroid.
  • Update centroids ‚Äì Recalculate the centroid of each cluster as the mean of all points in the cluster.
  • Repeat ‚Äì Continue steps 3-4 until convergence (centroids no longer change or move minimally).
